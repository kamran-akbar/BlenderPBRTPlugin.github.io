Introduction

  • Item Name : Physically-based Rendering Tool Blender Plugin
  • Item Version : v 1.0
  • Author : Filipe Gama, Kamran Akbar
  • License : MIT License

This plugin is meant for creating .pbrt scene file for the physically-based rendering engine. This plugin will provide you required user interface to define different paramteres inside the PBRT scene. In this documentation, you can read and comprehend each component and their properties. The definition of this properties are take from the PBRT website.

PBRT Sampler

PBRT Sampler is in the Render Properties.

Sampler

The Sampler generates samples for the image, time, lens, and Monte Carlo integration. A number of implementations are provided; the default "halton"â€”is an instance of the HaltonSampler that takes 16 samples per pixel.
Default: Halton

Samples per Pixel

The number of samples to take, per pixel. Note that the number of samples is taken per pixel on average; depending on the actual sampling algorithm being used, individual pixel areas may have slightly more or slightly fewer.
Default: 16
Range: [1, infinity]

PBRT Integrator

PBRT Integrator is in the Render Properties.

Integrator

The integrator implements the light transport algorithm that computes radiance arriving at the film plane from surfaces and participating media in the scene. The default integrator is the PathIntegrator.
Default: Path Integrator

Max Depth

Maximum length of a light-carrying path sampled by the integrator.
Default: 5
Range: [0, infinity]

PBRT Execution

PBRT Execution is in the Render Properties.

Path to PBRT Build Folder

This property is for inserting the path which obj2pbrt.exe exists.

Generate PBRT Scene File

By pressing this button .pbrt scene file will be created in PBRT_SceneFiles folder.

PBRT Random Scene Generator

PBRT Random Scene Generator is inside the Render Properties.

Path to 3D Meshes

This properties refers to the path where 3D meshes exists.

Object Number

This number define the amount of 3D meshes will be taken from the path mentioned above.
Default: 5
Range: [0, infinity]

X min and max

X min and X max define the minimum and maximum position range for X axis.
Default: X min = -1, X max = 1
Range: [-infinity, infinity]

Y min and max

Y min and Y max define the minimum and maximum position range for Y axis.
Default: Y min = -1, Y max = 1
Range: [-infinity, infinity]

Z min and max

Z min and Z max define the minimum and maximum position range for Z axis.
Default: Z min = -1, Z max = 1
Range: [-infinity, infinity]

Generate Random Scene

By Pressing this button, specified number of objects that is defined in Object Number property are chosen from the 3D mesh folder path which is defined in Path to 3D Meshes property.

Generate Random Material

By Pressing this button a PBRT Shader Node is created for each object in the scene with random properties.

PBRT Light

PBRT Light is in the Light Data Properties.

Light Type

Light sources cast illumination in the scene. There are two types of lights: point light and spot light.
Default: Point Light

Color

This property defines the light color in rgb channel.
Default: (1.0, 1.0, 1.0)
Range: [(0.0, 0.0, 0.0), (1.0, 1.0, 1.0)]

SPD

This property represents a file path to a file which consists the spectral power distribution. If this property is empty, the plugin will use Color Property

Power

Power modulates the amount of light that the light source emits into the scene.
Default: 10
Range: [0, infinity]

Goniometric Radiation(Point Light)

The filename of the image file that stores a goniometric diagram to use for the lighting distribution.

Cone Angle(Spot Light)

The angle that the spotlight's cone makes with its primary axis.
Default: 30
Range: [0, 180]

Cone Delta Angle(Spot Light)

The angle at which the spotlight intensity begins to fall off at the edges.
Default: 5
Range: [0, 180]

PBRT Camera

PBRT Camera is in the Camera Data Properties.

Camera Type

Two types of camera currently supported including Perspective and Orthographic.
Default: Perspective

Field of View

Specifies the field of view for the perspective camera. This is the spread angle of the viewing frustum along the narrower of the image's width and height.
Default: 90
Range: [0, 180]

Shutter Open

The time at which the virtual camera shutter opens.
Default: 0
Range: [0, infinity]

Shutter Close

The time at which the virtual camera shutter closes.
Default: 1
Range: [0, infinity]

Lens Radius

The radius of the lens. Used to render scenes with depth of field and focus effects. The default value yields a pinhole camera.
Default: 0
Range: [0, infinity]

Focal Distance

The focal distance of the lens. If lensradius is zero, this has no effect. Otherwise, it specifies the distance from the camera origin to the focal plane.
Default: 1
Range: [0, infinity]

PBRT Film

PBRT Film is in the Camera Data Properties.

Resolution X

The number of pixels in the X direction.
Default: 640
Range: [0, infinity]

Resolution Y

The number of pixels in the Y direction.
Default: 480
Range: [0, infinity]

Scale

Scale factor to apply to film pixel values before saving the image.
Default: 1
Range: [0, infinity]

Diagonal Length

Diagonal length of the film, in mm. (This value is only used when the RealisticCamera is used.
Default: 35
Range: [0, infinity]

PBRT Shader Node

PBRT Shader Node is in the Shader Editor->Add->PBRT Nodes.

Model

General material model that supports BSDF.
Default: Universal

Kr

The reflectivity of the surface.
Default: (0.5, 0.5, 0.5)
Range: [(0.0, 0.0, 0.0), (1.0, 1.0, 1.0)]

Kt

The transmissivity of the surface.
Default: (0.0, 0.0, 0.0)
Range: [(0.0, 0.0, 0.0), (1.0, 1.0, 1.0)]

k

Absorption coefficient to use in computing the material's reflectance.
Default: (0.0, 0.0, 0.0)
Range: [(0.0, 0.0, 0.0), (1.0, 1.0, 1.0)]

IoR

Object's index of refraction.
Default: 1.33
Range: [0, infinity]

Roughness

Material's roughness. Affects specular reflection and transmission.
Default: 0.5
Range: [0.0, 1.0]

Opacity

The opacity of the surface. Note that when less than one, the material transmits light without refracting it.
Default: 1.0
Range: [0.0, 1.0]

Ks

The coefficient of specular reflection and transmission.
Default: (1.0, 1.0, 1.0)
Range: [(0.0, 0.0, 0.0), (1.0, 1.0, 1.0)]

Kd

The diffuse reflectivity of the surface.
Default: 0.0
Range: [0.0, 1.0]

Anisotropic

Controls degree of anisotropy in the specular highlight.
Default: 0.0
Range: [0.0, 1.0]

U Roughness

Microfacet roughness in the u direction. If zero, perfect specular reflection is modeled.
Default: 0.5
Range: [0.0, 1.0]

V Roughness

Microfacet roughness in the v direction. If zero, perfect specular reflection is modeled.
Default: 0.5
Range: [0.0, 1.0]

Remap Roughness

If true, roughness values are expected to be in the range [0,1], and are remapped to microfacet distribution function parameter values that range from near-perfect-specular at 0 to very rough at 1. Otherwise the roughness parameters are used directly for the alpha parameters of the microfacet distribution function.
Default: True
Range: False or True

File Input Node

File Input Node is in the Shader Editor->Add->PBRT Nodes.

File Input

A path to the file defined inside this property.

Hyperspectral Texture Node

Hyperspectral Texture Node is in the Shader Editor->Add->PBRT Nodes.

File Input

A path to the hyperspectral texture file with .hdr format.

RGB File Name

The final sRGB texture name which its default value is equal to hdr file name.

PBRT Material Match

PBRT Material Match is in the Material Properties.

Material Match

This button will create a PBRT Shader Node, Principled BSDF, and Output Shader Node for the selected object if it does not have one and it matches up PBRT Shader Node's properties with Principled BSDF properties.
If Hyperspectral Texture Node is connected to the PBRT Shade Node's Kr Socket, Material Match button will transfrom the hyperspectral texture to sRGB texture and assigns it to Principled BSDF's base color via Image Texture Node. Besides, this button will create Mapping and Texture Coordinate node for users to change the sRGB texture scale and offset.
